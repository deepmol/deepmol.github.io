# 🧬 Quantum DeepMol

Интерактивная многопользовательская платформа для проектирования молекул *de novo* с использованием гибридных вычислений: классический ИИ + квантовый отжиг.

## 🚀 О проекте

Проект объединяет игровую интуицию (в стиле Foldit) с мощностью современных технологий для ускорения поиска новых лекарственных соединений.

### Ключевые технологии:

- **Мультиплеер:** Yjs — обеспечивает бесконфликтную совместную работу в реальном времени.
- **ИИ-ядро:** MolecularGPT — специализированные LLM для генерации химически валидных структур (SMILES/SELFIES).
- **Фреймворк:** DeepMol — интеграция машинного обучения и пайплайнов для предсказания свойств.
- **Квантовый бэкенд:** D-Wave — используется для поиска глобального минимума энергии молекул через квантовый отжиг.

## 🏗 Архитектура

Платформа разделена на две ключевые составляющие:

- **MolecularGPT:** Отвечает за генеративную часть (De Novo Design). Модель предлагает модификации на основе текстовых запросов или текущего состояния проекта.
- **DeepMol App:** Клиентское приложение и визуализатор. Здесь игроки взаимодействуют с молекулой, проводят «слияние» версий и ревью правок.

## 🎮 Игровые механики

- **Общий проект:** Совместное редактирование структуры белка или молекулы.
- **Текст в молекулу:** Генерация структур через запросы к ЯндексGPT / MolecularGPT.
- **Квантовая оптимизация:** Запуск задачи на D-Wave для уточнения геометрии связи в один клик.
