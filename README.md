# 🧬 Quantum DeepMol

Интерактивная многопользовательская платформа для проектирования молекул *de novo* с использованием гибридных вычислений: классический ИИ + квантовый отжиг.

## 🚀 О проекте

Проект объединяет игровую интуицию (в стиле Foldit) с мощностью современных технологий для ускорения поиска новых лекарственных соединений.

### Ключевые технологии:

- **Мультиплеер:** Yjs — обеспечивает бесконфликтную совместную работу в реальном времени.
- **ИИ-ядро:** MolecularGPT — специализированные LLM для генерации химически валидных структур (SMILES/SELFIES).
- **Фреймворк:** DeepMol — интеграция машинного обучения и пайплайнов для предсказания свойств.
- **Квантовый бэкенд:** D-Wave — используется для поиска глобального минимума энергии молекул через квантовый отжиг.

## 🏗 Архитектура

Платформа разделена на две ключевые составляющие:

- **MolecularGPT:** Отвечает за генеративную часть (De Novo Design). Модель предлагает модификации на основе текстовых запросов или текущего состояния проекта.
- **DeepMol App:** Клиентское приложение и визуализатор. Здесь игроки взаимодействуют с молекулой, проводят «слияние» версий и ревью правок.

## 🎮 Игровые механики

- **Общий проект:** Совместное редактирование структуры белка или молекулы.
- **Текст в молекулу:** Генерация структур через запросы к ЯндексGPT / MolecularGPT.
- **Квантовая оптимизация:** Запуск задачи на D-Wave для уточнения геометрии связи в один клик.

## 🌌 Основная концепция
Мы переносим опыт Foldit на новый уровень. Вместо одиночного решения задач, пользователи работают в едином пространстве над «Общим проектом», где ИИ предлагает варианты, а квантовый компьютер находит идеальные энергетические состояния.

## 🛠 Технологический стек
Проект базируется на четырех «китах»:

*   **Real-time Collaboration (Yjs):** Синхронизация действий игроков без конфликтов. Каждый атом, измененный одним участником, мгновенно отображается у всех.
*   **AI Engine (MolecularGPT):** Модели Deep Learning для генерации SMILES-структур и предсказания аффинности связывания.
*   **Core Framework (DeepMol):** Масштабируемая инфраструктура для обработки химических данных и визуализации.
*   **Quantum Annealing (D-Wave):** Использование квантового отжига для решения задач комбинаторной оптимизации при стыковке (docking) молекул.

## 🖥 Распределенные вычисления (DeepFoldit Network)
Проект использует возможности распределенной сети (Distributed Computing) для обработки тяжелых вычислений:

*   **Volunteer Computing:** Участники могут жертвовать свободные мощности своих GPU/CPU для обсчета динамики белков прямо через браузер или клиент (по аналогии с Folding@home).
*   **Shared Inference:** Распределенное выполнение запросов к моделям MolecularGPT, что снижает нагрузку на центральные серверы.
*   **Peer-to-Peer Training:** Коллективное дообучение моделей на основе успешных «игровых» паттернов, найденных сообществом.

## 🧬 Игровой процесс
*   **De Novo Design:** Введите текстовый запрос (например: «ингибитор для белка X»), и ИИ сгенерирует стартовую молекулу.
*   **Collaborative Folding:** Манипулируйте структурой вместе с командой.
*   **Quantum Optimization:** Нажмите кнопку «Сгенерировать», чтобы отправить сложную задачу по минимизации энергии в квантовое облако D-Wave.
*   **GitVerse Integration:** Сохраняйте версии своих молекул, делайте «Слияние» (Merge) удачных решений и проводите «Ревю» химической валидности.

## 📦 Структура репозиториев
*   **MolecularGPT** — Генеративные модели и ИИ-логика.
*   **DeepMol** — Фронтенд приложения, мультиплеер и интеграция с D-Wave.
