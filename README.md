# 🧬 Quantum DeepMol

Интерактивная многопользовательская платформа для проектирования молекул *de novo* с использованием гибридных вычислений: классический ИИ + квантовый отжиг.
Проект объединяет научную визуализацию, motion design и игровую механику (в стиле Foldit) для ускорения поиска новых лекарственных соединений.

## 🚀 О проекте

Среда, где исследователи и энтузиасты могут совместно конструировать молекулы, опираясь на подсказки ИИ и квантовые вычисления. <br> Визуализация и анимация играют ключевую роль: они превращают абстрактные химические данные в интуитивно понятные образы, помогая быстрее принимать решения и лучше понимать поведение молекул.

### Ключевые технологии

- **Мультиплеер:** Yjs — обеспечивает бесконфликтную совместную работу в реальном времени.
- **ИИ-ядро:** MolecularGPT — специализированные LLM для генерации химически валидных структур (SMILES/SELFIES).
- **Фреймворк:** DeepMol — интеграция машинного обучения, пайплайнов для предсказания свойств и визуализации.
- **Квантовый бэкенд:** D-Wave — используется для поиска глобального минимума энергии молекул через квантовый отжиг.
- **Научная визуализация и motion design:** плавные анимации, переходы и интерактивные эффекты, повышающие когнитивную ясность и пользовательский опыт.

## 🏗 Архитектура

Платформа разделена на две ключевые составляющие:

- **MolecularGPT:** Отвечает за генеративную часть (De Novo Design). Модель предлагает модификации на основе текстовых запросов или текущего состояния проекта.
- **DeepMol:** Клиентское приложение и визуализатор. Здесь игроки взаимодействуют с молекулой, видят анимированные изменения, проводят «слияние» версий и ревью правок.

## 🎮 Игровые механики

- **Общий проект:** Совместное редактирование структуры белка или молекулы.
- **Текст в молекулу:** Генерация структур через запросы к MolecularGPT.
- **Квантовая оптимизация:** Запуск задачи на D-Wave для уточнения геометрии связи в один клик.
- **Визуальная обратная связь:** Анимации, сопровождающие каждый этап — от генерации до квантового расчёта.

## 🌌 Основная концепция

Мы переносим опыт Foldit на новый уровень. Вместо одиночного решения задач пользователи работают в едином пространстве над «Общим проектом», где ИИ предлагает варианты, а квантовый компьютер находит идеальные энергетические состояния. Научная визуализация с элементами motion design делает этот процесс наглядным и увлекательным.

## 🛠 Технологический стек

Проект базируется на четырёх «китах»:

- **Real-time Collaboration (Yjs):** Синхронизация действий игроков без конфликтов. Каждый атом, изменённый одним участником, мгновенно отображается у всех.
- **AI Engine (MolecularGPT):** Модели Deep Learning для генерации SMILES-структур и предсказания аффинности связывания.
- **Core Framework (DeepMol):** Масштабируемая инфраструктура для обработки химических данных, визуализации и интеграции с квантовым бэкендом.
- **Quantum Annealing (D-Wave):** Использование квантового отжига для решения задач комбинаторной оптимизации при стыковке (docking) молекул.
- **Web-based 3D rendering:** `<model-viewer>` (на базе WebGL/Three.js) для интерактивного просмотра молекулярных структур.
- **Motion Design System:** стандартизированные анимации и переходы на основе CSS-переменных и токенов (длительности, easing-функции), обеспечивающие согласованность интерфейса и улучшающие пользовательский опыт.

## ✨ Motion Design и научная визуализация

Вдохновляясь современными исследованиями в области технической визуализации, мы внедрили motion design как неотъемлемую часть интерфейса. Анимации не просто украшают — они выполняют функциональные задачи:

- **Переключение 3D-моделей:** при генерации молекулы по текстовому запросу происходит плавный cross-fade переход между старой и новой структурой (длительность 500 ms, easing `ease-in-out`). Это помогает пользователю сохранить контекст и увидеть эволюцию модели.
- **Визуализация квантовой оптимизации:** во время расчёта на D-Wave атомы «дышат» (лёгкая вибрация), а по окончании — координаты плавно морфируют к новому энергетическому минимуму. Одновременно анимированный график энергии заполняется, отражая прогресс.
- **Инвентарь атомов:** при наведении ячейка слегка увеличивается (scale 1.05), при клике — распространяется ripple-эффект, а символ атома «влетает» в поле ввода. Это даёт тактильный отклик и повышает вовлечённость.
- **Git-таймлайн:** новые коммиты появляются с анимацией slide-down + fade-in, маркеры изменяют статус пульсацией. История разработки становится живой.
- **Боковые панели:** открываются с эффектом «пружины» (custom cubic-bezier), контент появляется с небольшой задержкой. Это предотвращает дезориентацию в многооконном интерфейсе.
- **Дисплей энергии:** значение изменяется с анимацией счётчика, цвет фона плавно переходит от зелёного к красному в зависимости от величины. Мгновенная, но мягкая обратная связь повышает доверие к данным.

Все анимации построены на системе **motion-токенов** — CSS-переменных, задающих единые длительности и кривые easing:

```css
:root {
  --motion-duration-fast: 200ms;
  --motion-duration-normal: 300ms;
  --motion-duration-slow: 500ms;
  --motion-easing-standard: cubic-bezier(0.4, 0.0, 0.2, 1);
  --motion-easing-spring: cubic-bezier(0.34, 1.56, 0.64, 1);
}
```

Это гарантирует визуальную согласованность во всём приложении и упрощает поддержку.  
Для пользователей с чувствительностью к анимации реализована поддержка `prefers-reduced-motion`, при которой все движения заменяются простыми затуханиями или отключаются.

Производительность достигается за счёт использования CSS-анимаций (где возможно) и оптимизации WebGL-рендеринга (`<model-viewer>` автоматически использует GPU). На мобильных устройствах сложные эффекты адаптируются или упрощаются.

## 🖥 Распределённые вычисления (DeepFoldit Network)

Проект использует возможности распределённой сети (Distributed Computing) для обработки тяжёлых вычислений:

- **Volunteer Computing:** Участники могут жертвовать свободные мощности своих GPU/CPU для обсчёта динамики белков прямо через браузер или клиент (по аналогии с Folding@home).
- **Shared Inference:** Распределённое выполнение запросов к моделям MolecularGPT, что снижает нагрузку на центральные серверы.
- **Peer-to-Peer Training:** Коллективное дообучение моделей на основе успешных «игровых» паттернов, найденных сообществом.

## 🧬 Игровой процесс

- **De Novo Design:** Введите текстовый запрос (например: «ингибитор для белка X»), и ИИ сгенерирует стартовую молекулу. Переход к новой модели будет плавным и наглядным.
- **Collaborative Folding:** Манипулируйте структурой вместе с командой. Действия каждого участника подсвечиваются анимацией.
- **Quantum Optimization:** Нажмите кнопку «Сгенерировать», чтобы отправить задачу по минимизации энергии в квантовое облако D-Wave. Наблюдайте за «дыханием» атомов и графиком энергии.
- **GitVerse Integration:** Сохраняйте версии своих молекул, делайте «Слияние» (Merge) удачных решений и проводите «Ревю» химической валидности — каждое изменение отражается в анимированном таймлайне.

## 📦 Структура репозиториев

- **MolecularGPT** — Генеративные модели и ИИ-логика.
- **DeepMol** — Фронтенд приложения, мультиплеер, интеграция с D-Wave, а также дизайн-система с motion-токенами и компонентами научной визуализации.
